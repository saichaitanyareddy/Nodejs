NPM:
npm is a package manager for the JavaScript programming language. 
It is the default package manager for the JavaScript runtime environment Node.js.

npm is written entirely in JavaScript and was developed by Isaac Z. Schlueter
Over 477,000 packages are available on the main npm registry
In npm version 6, the audit feature was introduced to help developers identify and fix vulnerability and security issues in installed packages.
